# ts-coverage

When you migrate your project to TypeScript it's useful to track the progress.
Ts-coverage print a simple histogram so you can always know how close to the goal you are.

## Example

`npx ts-coverage ./src`

*Output*

```
.js(x) | ################# | 1933 (97.9%)
.ts(x) | #                 | 41 (2.1%)
```
